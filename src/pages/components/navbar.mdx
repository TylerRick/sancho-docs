# Navbar

A Navbar is used to provide app based navigation.

#### Basic usage

You'll typically use the Navbar in combination with a [Toolbar](/components/toolbar). You can supply a custom background colour using standard styling, as in the example below.

```.jsx
<Navbar>
  <Toolbar>
    {/* Toolbar content goes here */}
  </Toolbar>
</Navbar>
```
